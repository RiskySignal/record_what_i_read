# Attack & Defense on Speech Recognition

### About

这个文件夹中归类了语音识别领域对抗攻击的相关工作，其中：

- `model.md` 中记录有关 ASR 模型的相关工作；

- `attack.md` 中记录有关对抗攻击的相关工作；
